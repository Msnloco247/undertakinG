# 🌐 UndertakingG — Frontend (Astro)

Esta es la interfaz de usuario de **UndertakingG**, diseñada para ofrecer una experiencia de planificación de negocios intuitiva, profesional y visualmente impactante.

---

## 🚀 Experiencia de Usuario

La interfaz está optimizada para guiar al emprendedor a través de la recolección de datos y la visualización de resultados estratégicos.

### Características Destacadas:
- **Cuestionarios Inteligentes**: Formularios validados que aseguran la calidad de la entrada para los modelos de IA.
- **Visualización "Ledger Blueprint"**: Un estilo visual inspirado en planos técnicos y libros contables de alta gama, utilizando un contraste nítido y tipografía de precisión.
- **Arquitectura Asíncrona**: Recupera y muestra múltiples secciones del plan de negocio en paralelo para minimizar la espera.
- **Componentes Reactivos**: Uso de islas de interactividad donde es necesario, manteniendo el resto del sitio estático y rápido.

---

## 🎨 Identidad Visual

El diseño de UndertakingG se basa en la autoridad y la precisión:
- **Tipografía**: `DM Serif Display` (Titulares con autoridad) y `JetBrains Mono` (Datos técnicos y precisos).
- **Paleta de Colores**: Fondo profundo con acentos en dorado y blanco técnico para resaltar métricas críticas.
- **Estructura**: Divisores táctiles y cuadrículas que evocan la organización de un plan de negocios maestro.

---

## 🛠️ Stack Tecnológico

- **Framework**: [Astro](https://astro.build/)
- **Estilos**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Lenguaje**: TypeScript
- **Iconografía**: Lucide React / Tabler Icons

---

## ⚙️ Desarrollo Local

1.  Instala las dependencias:
    ```bash
    bun install
    ```
2.  Inicia el servidor de desarrollo:
    ```bash
    bun run dev
    ```
3.  Construye para producción:
    ```bash
    bun run build
    ```
