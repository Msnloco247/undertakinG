# UndertakingG — Frontend (Astro)

Esta es la interfaz web interactiva de la plataforma **UndertakingG**. Proporciona un proceso fluido de recolección de datos para emprendedores y renderiza análisis de negocio detallados utilizando el sistema de diseño especializado "Ledger Blueprint".

## 🚀 Características Clave

- **Cuestionarios Intuitivos**: Formularios inteligentes con contadores de caracteres y validación en tiempo real.
- **Visualización Ledger Blueprint**: Un diseño técnico de alto contraste que visualiza planes de negocio con claridad financiera.
- **Obtención de Resultados en Paralelo**: Recuperación de datos asíncrona para mostrar múltiples análisis estratégicos simultáneamente.
- **Carga con Skeleton Screens**: Cargadores estructurados que reflejan los resultados finales para mejorar el rendimiento percibido.

## 🛠️ Stack Tecnológico

- **Framework**: Astro (SSG/SSR)
- **Lenguaje**: TypeScript
- **Estilos**: CSS puro con enfoque en tipografía y estética de alto contraste.
- **Estado**: Gestión de formularios en el lado del cliente y renderizado de resultados asíncronos.

## 🎨 Principios de Diseño

La interfaz sigue una estética de "Ledger Blueprint":
- **Tipografía**: `DM Serif Display` para autoridad y `JetBrains Mono` para precisión técnica.
- **Separación**: Divisores de alto contraste y reglas verticales discontinuas para agrupamiento lógico.
- **Enfoque**: Uso estratégico de acentos dorados para resaltar métricas críticas de presupuesto y crecimiento.

## 🛠️ Comandos

Desde el directorio `frontend`:
- `bun install`: Instala las dependencias.
- `bun run dev`: Inicia el servidor de desarrollo local.
- `bun run build`: Compila para producción.
